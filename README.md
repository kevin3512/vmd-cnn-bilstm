通过VMD-CNN-BiLSTM组合模型预测电力行业单日碳排放量

针对电力行业碳排放序列的非平稳性、多尺度及非线性特征导致的预测精度不足问题，本研究提出一种基于变分模态分解（VMD）与频域分量自适应预测的混合集成模型。首先，利用VMD技术将原始碳排放序列分解为一系列具有不同中心频率的本征模态分量（IMFs）。依据分量特性，将其归纳为表征随机噪声与短期波动的高频分量、反映季节性与周期性规律的中频分量以及表征长期趋势的低频分量。针对高频分量噪声显著的特点，构建卷积神经网络与长短期记忆网络融合模型进行预测，以充分捕捉其局部特征与序列依赖。对于规律性较强的中频分量，采用季节自回归积分滑动平均模型进行精准拟合；对于平滑的低频趋势分量，则利用循环神经网络进行建模。最后，将各分量的预测结果进行线性重构，得到碳排放的最终预测值。基于实际电力行业碳排放数据的实验表明，该模型在均方根误差（RMSE）和平均绝对百分比误差（MAPE）等关键指标上，相较于单一预测模型及未引入降噪的分解集成模型均有显著提升，验证了其对于提升电力碳排放预测精度的有效性。
